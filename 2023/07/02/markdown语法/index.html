    Markdown基础语法

    发表于 2023-07-02 更新于 2023-07-04 本文字数： 1.1k 阅读时长 ≈ 4 分钟

    Markdown是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与HTML混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。因简洁、高效、易读、易写，Markdown被大量使用，如Github、Wikipedia、简书等。

    markdown支持HTML语法，这里仅就markdown语法进行说明

    标题语法

    有多少个#就代表着几级标题

    这是一级标题 #

    这是二级标题 ##

    段落语法

    要创建段落，请使用空白行或多行文本进行分割

    换行语法

    在一行的末尾添加两个或多个空格，然后按回车键，即可创建一个换行几乎每个Markdown都支持两个或多个和空格进行换行,这样可能会导致兼容性问题建议换行使用 <br>

    粗体语法

    ** 你的文字 **，左右两边各两个*可以使字体加粗
    下面是例子:
    请问您今天要来点兔子吗?
    同时，你也可以用__你的文字__来加粗，不建议这样使用。Markdown 应用程序在如何处理单词或短语中间的下划线上并不一致。为兼容考虑，在单词或短语中间部分加粗的话，请使用星号（asterisks）

    引用语法

    要创建块引用，请在段落前添加一个 > 符号。

    注意，创建块引用必须在段落前！

    列表语法

    要创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点。数字不必按数学顺序排列，但是列表应当以数字 1 起始。
    例如：
    手机厂商出货量排行
    1.Samsung
    2.Apple
    3.Xiaomi
    4.OPPO
    要创建无序列表，请在每个列表项前面添加破折号 (-)、星号 (*) 或加号 (+) 。缩进一个或多个列表项可创建嵌套列表。
    例如：
    高级编程语言列表(节选)

    • -Python
    • -C
      或者
    • *Python
    • *C
      或者
    • +Python
    • +C

    在列表中嵌套其他元素

    要在保留列表连续性的同时在列表中添加另一种元素，请将该元素缩进四个空格或一个制表符，如下例所示：

    • This is the first list item.
    • Here’s the second list item.

    I need to add another paragraph below the second list item.

    • And here’s the third list item

    代码语法

    用“`”来包围住代码段即可，如果你要表示为代码的单词或短语中包含一个或多个反引号，则可以通过将单词或短语包裹在双反引号
    例如：

    print("HelloWorld!")

    Markdown基本语法允许您通过将行缩进四个空格或一个制表符来创建代码块。如果发现不方便，请尝试使用受保护的代码块。根据Markdown处理器或编辑器的不同，您将在代码块之前和之后的行上使用三个反引号（```）或三个波浪号（~~~）。

    分割线语法

    要创建分隔线，请在单独一行上使用三个或多个星号 (***)、破折号 (—) 或下划线 (___) ，并且不能包含其他内容。三个和三个以上显示的效果是一样的
    例如：

    为了兼容性，请在分隔线的前后均添加空白行。

    链接语法

    链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。超链接Markdown语法代码:

    [超链接显示名](超链接地址 “超链接title”)
    哔哩哔哩

    给链接增加 Title

    链接title是当鼠标悬停在链接上时会出现的文字，这个title是可选的，它放在圆括号中链接地址后面，跟链接地址之间以空格分隔。

    哔哩哔哩

    网址和Email地址

    用<>可以很简单的让http和邮箱地址变成可点击的链接

    https://b23.tv/WnMIYnG
    fuibafuyu@qq.com

    图片语法

    要添加图像，请使用感叹号 (!), 然后在方括号增加替代文本，图片链接放在圆括号里，括号里的链接后可以增加一个可选的图片标题文本。

    如果你想增加图片跳转链接功能，则用[]包住图片链接再在后面用(‘href’)即可
